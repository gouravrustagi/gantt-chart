# Gantt Chart Generator - Mobile & Desktop App

A responsive web application for creating Gantt charts that works on both mobile and desktop devices.

## Features
- ✅ Mobile-responsive design
- ✅ 5 different chart types
- ✅ Duration in days, hours, and minutes
- ✅ Real-time task management
- ✅ Works on any device with a browser

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the web app:
```bash
python app.py
```

3. Open in browser:
- **Desktop**: http://localhost:5000
- **Mobile**: http://YOUR_IP_ADDRESS:5000

## Usage on Mobile

1. Connect your phone to the same WiFi network as your computer
2. Find your computer's IP address
3. Open browser on phone and visit: http://YOUR_IP:5000
4. Add tasks and generate charts!

## Desktop App

To use the desktop version:
```bash
python ganttchrt.py
```

## Chart Types
1. Standard Gantt - Classic bar chart
2. Timeline View - Dates on bars
3. Milestone Chart - Marker-based view
4. Compact View - Minimal design
5. Waterfall View - Progress tracking
